# How Engineers Think

A collection of real-world engineering lessons explained with small, practical code examples.

This repository documents how I’m learning to think, make decisions, and take responsibility as an engineer—focusing on judgment, trade-offs, and long-term code quality.

## What’s Inside
- Engineering principles from real scenarios  
- Minimal Python and C++ examples  
- Emphasis on failure handling and system thinking  

## Tech
- Next.js 14
- TypeScript
- Clean Architecture
- Secure server-side code execution

## Run Locally
```bash
npm install
npm run dev
```
